## Step-by-Step Instructions

### **STEP 1: Install FlutterFire CLI** (One-time setup)
```bash
dart pub global activate flutterfire_cli
```

### **STEP 2: Login to Firebase**
```bash
firebase login
```

### **STEP 3: Configure Firebase for Your Project**
Run this command in your project root directory:
```bash
flutterfire configure
```

This will:
- Detect your Flutter project
- Let you select/create a Firebase project
- Generate `firebase_options.dart` automatically
- Configure for Android, iOS, Web, etc.

### **STEP 4: Install Dependencies**
```bash
flutter pub get
```

### **STEP 5: Update main.dart**
I've already updated your `main.dart` file to initialize Firebase. Make sure it looks like this:

```dart
import 'package:flutter/material.dart';
import 'package:firebase_core/firebase_core.dart';
import 'package:nak_electronics/core/firebase/firebase_service.dart';
import 'firebase_options.dart'; // Generated by flutterfire configure
import 'features/home/homepage.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(
    options: DefaultFirebaseOptions.currentPlatform,
  );
  runApp(const MyApp());
}
```

### **STEP 6: Android Configuration**
- The `google-services.json` file will be automatically added to `android/app/`
- Make sure your `android/app/build.gradle` has:
  ```gradle
  apply plugin: 'com.google.gms.google-services'
  ```
- And `android/build.gradle` has:
  ```gradle
  classpath 'com.google.gms:google-services:4.4.0'
  ```

### **STEP 7: iOS Configuration** (if needed)
- The `GoogleService-Info.plist` will be automatically added to `ios/Runner/`
- Make sure it's added to your Xcode project

### **STEP 8: Test Firebase Connection**
Create a test to verify Firebase is working:

```dart
// In your code, you can now use:
FirebaseService.auth.signInAnonymously();
FirebaseService.firestore.collection('test').add({'test': 'data'});
```
